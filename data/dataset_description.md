Dataset 1: Ethereum Fraud Detection Dataset

Source:
https://www.kaggle.com/datasets/vagifa/ethereum-frauddetection-dataset

Description:
This dataset contains transaction-level features of Ethereum wallets used for fraud detection. It includes behavioral and financial attributes of each wallet.

Number of Instances:
9842

Number of Attributes:
50

Target Variable:
FLAG (0 = Legitimate, 1 = Fraud)

How to Download:
Go to Kaggle website
Search "Ethereum Fraud Detection Dataset"
Download the dataset
Place the file inside the data/ folder

Dataset 2: Bitcoin Heist Dataset

Source:
https://archive.ics.uci.edu/dataset/526/bitcoin+heist+ransomware+address+dataset

Description:
This dataset contains Bitcoin transaction graph features used to identify ransomware addresses. Each row represents a wallet address with network-based features.

Number of Instances:
~2.9 million (sampled subset used in project)

Number of Attributes:
10 (including label)

Target Variable:
label (white = Legitimate, others = Ransomware/Fraud)

How to Download:
Visit the UCI dataset link above
Download the dataset
Place the file inside the data/ folder
