Dataset 1: Ethereum Fraud Detection Dataset

Source:
https://www.kaggle.com/datasets/vagifa/ethereum-frauddetection-dataset

Description:
This dataset contains transaction-level features of Ethereum wallets used for fraud detection. It includes behavioral and financial attributes of each wallet.

Number of Instances:
9842

Number of Attributes:
50

Target Variable:
FLAG (0 = Legitimate, 1 = Fraud)

How to Download:
Go to Kaggle website
Search "Ethereum Fraud Detection Dataset"
Download the dataset
Place the file inside the data/ folder
